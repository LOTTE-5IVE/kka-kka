## 🍪 Intro 

**까까**는 기존의 도매 중심 전문몰과 종합몰 중심의 간식 시장 문제 해결을 위한 **간식 전문 버티컬 플랫폼**입니다.

개인 소비자의 욕구를 충족하고 나아가 현명한 소비를 도울 수 있도록

✔️ 구매 이력에 따른 추천

✔️ 검색 기능 고도화

✔️ 프로모션 노출 확대

와 같은 **사용자 맞춤형 서비스**를 제공합니다.
